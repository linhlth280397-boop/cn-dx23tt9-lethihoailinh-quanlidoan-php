# Website Quản Lý Đồ Án Sinh Viên

## Giới thiệu
Đây là hệ thống website quản lý đồ án sinh viên tại Khoa Công nghệ Thông tin, giúp sinh viên đăng ký đề tài và giảng viên quản lý, đánh giá đồ án.

## Chức năng chính
### Sinh viên
- Đăng ký / đăng nhập
- Xem danh sách đề tài
- Đăng ký đề tài

### Giảng viên
- Tạo đề tài
- Xem sinh viên đăng ký
- Duyệt đề tài

## Công nghệ sử dụng
- PHP
- MySQL
- HTML, CSS, JavaScript
